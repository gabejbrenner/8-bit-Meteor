About

8-bit-meteor is an action/platformer style game, sort of like a mix between Contra, Mario, and Super Metroid. I am currently developing in Jruby, using LWJGL with Slick, and the Tiled Map Editor but when I get to a stable alpha release I will include a compiled Jar.

Where We are Now

I have not currently reached a stable alpha version, I have spent most of my time working on data APIs and have not written any real game content yet. Before I am comfortable enough to compile though I have a small Checklist.

  • ☐ Create Main Spritesheet
  • ☐ Implement Spritesheet API
  • ☐ Implement Collision Detection
  • ☐ Implement Gravity
  • ☐ Implement Jumping
  • ☑ Implement Tiled Map API
  • ☑ Implement Map Rendering API
  • ☑Implement Simple Level Data API
